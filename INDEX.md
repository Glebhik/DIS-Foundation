## How to Use This Repository

1. **Find knowledge** — Navigate by domain above, or use GitHub's search.
2. **Add knowledge** — Create a document in the right domain folder, then link it from the section `README.md`.
3. **Update knowledge** — Edit in place; for major rewrites, archive the old version first.
4. **Remove knowledge** — Never delete. Move to `/archive` with a date suffix and a superseded note.
