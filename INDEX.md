## How to Use This Repository

| Intent | Action |
|---|---|
| **Find knowledge** | Navigate by domain above, or use GitHub's search |
| **Add knowledge** | Create in the right domain → link from section `README.md` → update this index |
| **Update knowledge** | Edit in place; for major rewrites, archive the old version first |
| **Remove knowledge** | Never delete — archive to `/archive/[domain]/` with a date suffix |
