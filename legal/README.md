# Legal

Contracts, compliance frameworks, IP policy, and legal reference documents.

## Contents

| File / Folder | Purpose |
|---|---|
| `contracts/` | Contract templates and signed agreements index |
| `compliance/` | Regulatory compliance frameworks (GDPR, local law) |
| `ip.md` | Intellectual property policy |
| `privacy.md` | Privacy policy and data handling rules |
| `nda-template.md` | Standard NDA template |

## Guidelines

- Do not store executed contracts with sensitive PII in this repo — link to secure storage instead.
- Compliance documents must cite the applicable regulation and last review date.
