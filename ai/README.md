# AI

DIS Group's AI strategy, tooling, models, prompts, agents, and internal AI development.

## Contents

| File / Folder | Purpose |
|---|---|
| `strategy.md` | AI strategy and adoption roadmap |
| `models.md` | Approved models and usage guidelines |
| `agents/` | Internal AI agents — specs and prompts |
| `prompts/` | Shared prompt library |
| `governance.md` | AI ethics, review process, acceptable use |
| `experiments/` | AI experiments and evaluations |

## Guidelines

- All production AI agents must have a spec in `agents/`.
- Prompt engineering decisions should be versioned — never overwrite, append new versions.
- Governance doc is mandatory reading before deploying any AI-facing feature.
