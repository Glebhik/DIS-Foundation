- Asset data must include capacity (MW/MWh), commissioning date, and current status.
- Regulatory documents must reference jurisdiction, filing date, and expiry.
- Link to `/esg` for carbon accounting, `/finance` for project economics, `/legal` for PPAs and grid agreements.
