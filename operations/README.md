# Operations

Day-to-day operating procedures, workflows, tools, and internal processes.

## Contents

| File / Folder | Purpose |
|---|---|
| `processes/` | Standard operating procedures (SOPs) |
| `tools.md` | Approved internal tools and access guide |
| `meetings/` | Meeting cadences, agendas, templates |
| `hiring/` | Recruitment process and job description templates |
| `vendor-management.md` | Vendor onboarding and review process |

## Guidelines

- SOPs use the format: Purpose → Trigger → Steps → Owner → Review Date.
- Link to `/legal` for contract-related processes, `/finance` for approvals.
