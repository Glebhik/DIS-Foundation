# Products

Documentation for all DIS Group products and services — specs, roadmaps, and release notes.

## Contents

| File / Folder | Purpose |
|---|---|
| `overview.md` | Portfolio overview |
| `[product-name]/` | One folder per product |
| `[product-name]/spec.md` | Product specification |
| `[product-name]/roadmap.md` | Product roadmap |
| `[product-name]/releases/` | Release notes by version |

## Guidelines

- Each product has its own subfolder.
- Link to `/technology` for technical architecture, `/marketing` for positioning.
