# Finance

Financial policies, budgets, reporting, and approval workflows.

## Contents

| File / Folder | Purpose |
|---|---|
| `policy.md` | Financial policy and controls |
| `budgets/` | Annual and departmental budgets |
| `reporting/` | Monthly and quarterly financial reports |
| `approvals.md` | Spending approval thresholds and authority matrix |
| `templates/` | Invoice, PO, and expense report templates |

## Guidelines

- Budget documents are confidential — mark with `[CONFIDENTIAL]` header.
- All approval thresholds require sign-off from the CFO or designated delegate.
