# Marketing

Brand guidelines, messaging, campaigns, content strategy, and market presence.

## Contents

| File / Folder | Purpose |
|---|---|
| `brand/` | Brand guidelines, logos, color palette, typography |
| `messaging.md` | Core messaging framework and value propositions |
| `campaigns/` | Campaign briefs and post-mortems |
| `content/` | Content calendar, blog drafts, social templates |
| `assets/` | Index of marketing assets (link to external storage) |

## Guidelines

- Brand assets are canonical — never modify without Design review.
- Campaign briefs require owner, budget, KPIs, and target audience before launch.
