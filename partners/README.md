# Partners

Partner ecosystem — integrations, alliances, resellers, and joint initiatives.

## Contents

| File / Folder | Purpose |
|---|---|
| `overview.md` | Partner ecosystem map |
| `[partner-name]/` | One folder per strategic partner |
| `[partner-name]/brief.md` | Relationship summary, contacts, terms |
| `[partner-name]/initiatives/` | Joint projects and agreements |
| `onboarding.md` | Partner onboarding checklist |

## Guidelines

- Each active partner has a folder with an up-to-date brief.
- Inactive or former partners are moved to `/archive/partners/`.
