# Travel

Corporate travel policy, booking guides, expense templates, and destination briefings.

## Contents

| File / Folder | Purpose |
|---|---|
| `policy.md` | Corporate travel policy |
| `booking.md` | How to book travel and accommodations |
| `expenses.md` | Expense submission guidelines and limits |
| `destinations/` | City and country briefings for frequent destinations |
| `templates/` | Expense report and itinerary templates |

## Guidelines

- Policy documents must reference the Finance approval chain.
- Destination briefings include visa requirements, safety notes, and key contacts.
