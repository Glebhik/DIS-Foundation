# Archive

Superseded documents, closed projects, former partner briefs, and historical records.

## Structure

Mirror the top-level folder structure:

```
archive/
  company/
  strategy/
  products/
  partners/
  ...
```

## Guidelines

- Never delete — archive instead. Append the archival date to the filename: `strategy-2024.md`.
- Add a one-line note at the top of archived documents: `> Archived: YYYY-MM-DD. Superseded by [link].`
- The archive is read-only by convention. No documents are edited here after archival.
