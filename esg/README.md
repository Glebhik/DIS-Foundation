# ESG

Environmental, Social, and Governance framework for DIS Group — reporting, targets, policies, and compliance.

## Contents

| File / Folder | Purpose |
|---|---|
| `framework.md` | DIS Group ESG framework and materiality assessment |
| `environmental/` | Carbon accounting, emissions data, reduction targets |
| `social/` | People metrics, community impact, supply chain standards |
| `governance/` | Board composition, policies, audit and compliance |
| `reporting/` | Annual ESG reports and disclosures |
| `ratings/` | ESG rating agency assessments and responses |
| `targets.md` | Committed ESG targets with timelines |

## Guidelines

- All environmental data must cite measurement methodology and baseline year.
- Targets must be SMART (Specific, Measurable, Achievable, Relevant, Time-bound).
- Link to `/recycling` and `/energy` for operational environmental data.
- Reports follow the disclosure standards in use (GRI, SASB, TCFD — specify which).
